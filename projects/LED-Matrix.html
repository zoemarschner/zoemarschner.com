LED Matrix

This past summer I taught myself about electrical engineering by creating an LED matrix from basic components. The hardware consists of an 11 by 11 grid of LEDs, an Arduino micro, and two SIPO shift registers. The software is written in the Arduino programming language, and uses multiplexing in conjunction with the shift registers in order to control the entire matrix from 14 pins of the Arduino.

In order to use the matrix as a clock display, I’m currently working on providing Wi-Fi connectivity for this project. I’m doing this using an ESP8266 module and code I’ve written in the Arduino programming language that sends AT commands over Serial to the Wi-Fi Module and uses UDP to obtain time data from an NTP server.
