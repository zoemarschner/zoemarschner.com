Seifert Surface Generator

View on GitHub

As part of a group project for Ithaca High School/Cornell University math seminar, I created a program in Python which takes a graphical representation of a knot and computes the Seifert Surface, genus, and Alexander Polynomial of the knot.

The Seifert Surface of a knot K is a surface that has a boundary equal to K. This program uses Seifert’s algorithm to construct a representation of the surface in .obj file format and also includes the necessary code to view this file using OpenGL in Python.

Alexander’s Polynomial is calculated using the original algorithm proposed in Alexander’s 1928 paper, and upper and lower bounds for the genus of the knot are calculated from the Seifert Surface and Alexander Polynomial.
